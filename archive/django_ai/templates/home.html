Welcome

This website consists of several games that I built. Each game is played against a computer which has played each game millions of times in order to learn the best strategies for play. I have not taught the computer what to look for - it only knows when it has won, lost, or tied. There are no heuristic methods or identifying traits, so the computer stores each configuration and the decisions it made in that configuration. For games that require too much data to store each configuration, all possible decisions, and the results of those decisions, I use bit folding to shrink the data. Different bit folds play against one another and the more successful ones are used here.

Have fun!
